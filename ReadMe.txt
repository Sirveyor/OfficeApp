This is a project for a land surveying office. This project does not address billing.
This project will provide the following features.
1.  A simple user interface for the surveying office surveying department to manage their survey projects.
2.  To manage their survey crews.
3   To manage their survey equipment.
4.  To manage their survey personel.
5.  To track crew time spent on projects.
6.  To track office drafting time.

This project will use a multi-user database and will access an excell file where all project information is stored.
This project will include a project manager to keep track of schedu1ing, work time and status. 
It will include a file manager to track fille access.
It will have report and print functions.
